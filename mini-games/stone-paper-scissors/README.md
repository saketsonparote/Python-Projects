✊ Stone Paper Scissors Game (Python)

A simple terminal-based game where you play Stone, Paper, Scissors against the computer.
First to score 5 points wins the game.

🧠 Concepts Used

while loop
conditional statements (if-elif-else)
score tracking
random module
user input

🎮 How It Works

You choose:

1 → Stone
2 → Paper
3 → Scissors
The computer randomly selects its move.
The winner of each round gets 1 point.

First to reach 5 points wins the game.

▶ How to Run
Open terminal in this folder and run:
python stone_paper.py
