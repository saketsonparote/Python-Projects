🎯 Number Guessing Game (Python)

A simple terminal-based game where the computer randomly selects a number between 1 and 100, and the player tries to guess it.

🧠 Concepts Used

while loop
if-elif-else
random module
user input handling
basic game logic

🚀 How It Works

The computer generates a random number.
You enter a guess.
The program tells you:
Go higher 📈
Go lower 📉
It counts how many attempts you took.

When guessed correctly — you win 🎉

▶ How to Run
Open terminal in this folder and run:
python number_guessing.py
